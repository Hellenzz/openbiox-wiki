## 如何修改并提交贡献

请fork组织仓库[openbiox-wiki](https://github.com/openbiox/openbiox-wiki), 修改README.Rmd文件，并在本地执行render.R文件，确认无误后提交Pull requests，审核通过即可合并更新。
